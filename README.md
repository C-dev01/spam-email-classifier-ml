# Spam Email Classifier

A mini Machine Learning project that classifies emails as **spam** or **ham** using the Multinomial Naive Bayes algorithm.

## Files
- `Spam_Email_Classifier.ipynb` — Google Colab notebook (main)
- `Spam_Email_Classifier.py` — optional Python script
- `Project_Report.pdf` — 1-page project report
- `screenshots/` — output screenshots

## How to run
Open the Colab notebook or run `Spam_Email_Classifier.py` in a Python environment with `pandas` and `scikit-learn` installed.
